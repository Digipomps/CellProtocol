# Security Policy

## Reporting Security Issues

Please do not disclose suspected vulnerabilities publicly before the HAVEN Ward
has had a reasonable opportunity to review and coordinate a fix.

Report security issues to the current HAVEN Ward, Stiftelsen Digipomps.

Preferred contact:

```text
security@digipomps.org
```

If this address is not yet active, use the official contact channel published
by Stiftelsen Digipomps.

## Scope

Security reports may include:

- vulnerabilities in HAVEN protocol implementations;
- identity, vault, signing, bridge, resolver, or access-control issues;
- flaws that expose private data or cross identity boundaries;
- vulnerabilities in official HAVEN hosted services;
- supply-chain or dependency risks affecting official HAVEN releases.

DiMy-specific security issues should follow DiMy's own security process unless
the issue crosses into HAVEN code or official HAVEN services.
